# world-models — brutalist design

A neo-brutalist / Swiss rendering of the [world-models](https://github.com/shehio/world-models) project — AlphaZero-strength chess & Go by distilling Stockfish / KataGo on a single GPU.

**Live demo:** https://shehio.github.io/world-models-brutalist/

Static, fully responsive (mobile + desktop), no build step. One of five distinct design treatments of the same content.
